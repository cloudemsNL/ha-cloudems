Stap 3 van 5
Automatisch inloggen instellen
Apparaten op je thuisnetwerk kunnen HA openen zonder wachtwoord. Dit doe je één keer in HA's configuratie.
⚠️ Let op
Dit geldt alleen voor apparaten op je thuisnetwerk (WiFi thuis). Van buitenaf (4G, ander WiFi) blijft de normale login gewoon werken.
  • Ga in HA naar Instellingen → Systeem → Bewerken configuratie of open het bestand configuration.yaml in een editor.
  • Kopieer de onderstaande code en plak hem helemaal bovenaan in het bestand (vóór eventuele andere regels).
  • Vervang UW_GEBRUIKER_ID door jouw eigen ID (zie tip hieronder).
  • Sla het bestand op en herstart Home Assistant.
💡 Jouw gebruikers-ID vinden
Ga naar Instellingen → Gebruikers → klik op jouw naam.
In de URL staat je ID: /config/users/dit-lange-getal-is-jouw-id
Kopieer dat lange getal.
Plak hier je gebruikers-ID (optioneel, voor live preview)
# Zet dit BOVENAAN je configuration.yaml homeassistant: auth_providers: - type: trusted_networks trusted_networks: - 192.168.1.0/24 # jouw thuisnetwerk trusted_users: 192.168.1.0/24: - UW_GEBRUIKER_ID allow_bypass_login: true - type: homeassistant # normaal login blijft werken
📡 Ander netwerk-adres?
192.168.1.0/24 is het meest gebruikelijk. Twijfel je? Ga op je tablet naar WiFi-instellingen en kijk wat het IP-adres begint met. Begint het met 192.168.0? Pas dan de code aan naar 192.168.0.0/24.
