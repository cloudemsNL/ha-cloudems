## Features
- 🧠 **NILM** — Non-Intrusive Load Monitoring (database + lokale AI + cloud AI)
- ⚡ **Per-fase stroom begrenzer** (solar, accu, EV)
- ☀️ **Zonne-dimmer** bij negatieve energieprijzen
- 💰 **Dag-ahead energieprijzen** (ENTSO-E, NL/BE/DE/FR/...)
- 📊 **Lovelace dashboard kaart**
- 🔋 **Slim EV laden** op basis van solar + fase capaciteit

## Installatie (HACS)
1. HACS → Aangepaste repositories → `https://github.com/cloudems/ha-cloudems`
2. Installeer CloudEMS
3. Herstart Home Assistant
4. Instellingen → Integraties → CloudEMS toevoegen
